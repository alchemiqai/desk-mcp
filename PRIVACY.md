# Privacy

The Alchemiq News Intelligence plugin connects Claude Code to the hosted Alchemiq Desk MCP server. Use of the plugin is governed by Alchemiq's published policies.

- **Privacy policy**: <https://alchemiq.ai/privacy>
- **Terms of service**: <https://alchemiq.ai/terms>

## What this plugin sends

When you invoke a tool, Claude Code transmits the tool name, the arguments you (or Claude on your behalf) provide, and your OAuth-issued session token to `https://mcp.alchemiqai.com/mcp`. Nothing from this repository collects or stores data on its own.

## Authentication

Sign-in is handled via secure OAuth 2.0 against Alchemiq's identity provider (Frontegg). Your credentials are never exchanged with Claude or with this repository.

## Contact

For privacy questions, contact liad@alchemiq.ai or see <https://alchemiq.ai/privacy>.
